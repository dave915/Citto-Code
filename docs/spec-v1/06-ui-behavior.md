# 06. UI 동작 명세

## 1) 입력창 동작
- Enter: 전송
- Shift+Enter: 줄바꿈
- textarea 자동 높이 조절 (최대 높이 제한)
- 전송 가능 조건
  - 텍스트 trim 길이 > 0 또는 첨부파일 >= 1
  - `isStreaming === false`
  - disabled 상태가 아님

## 2) IME(한글/일본어) 보호
- 조합 중 Enter 전송 금지
- 조합 종료 직후 짧은 윈도우 Enter 무시
- keyCode 229 케이스 보호

## 3) 메시지 렌더링
- User: 우측 버블 + 첨부 파일 칩
- Assistant: 좌측 버블 + Markdown(GFM)
- 코드 블록: 하이라이트 스타일 적용
- 툴 호출: 별도 블록으로 실행 상태/결과 표시

## 4) 사이드바 동작
- 새 세션 생성
- 세션 전환
- 세션 삭제(최소 1개 세션 유지)
- 세션별 폴더 변경
