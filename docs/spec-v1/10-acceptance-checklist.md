# 10. 수용 기준 체크리스트

1. 새 세션 생성/전환/삭제가 정상 동작한다.
2. Enter 전송과 Shift+Enter 줄바꿈이 분리 동작한다.
3. 한글 입력 중 마지막 글자 중복 전송이 재현되지 않는다.
4. 스트리밍 중 중단 버튼이 즉시 반응한다.
5. tool-start/tool-result가 같은 메시지 블록에 표시된다.
6. 파일 첨부 전송 시 prompt 조립 규칙이 일치한다.
7. 모델/권한/플랜 모드가 CLI 인자에 반영된다.
8. 오류 배너가 표시되고 이후 재전송이 가능하다.
9. 연속 요청에서도 프로세스 누수 없이 동작한다.
10. 빌드 명령이 성공한다.

## 권장 검증 시나리오
- 기본 텍스트 전송 3회 연속
- 파일 2개 첨부 + 질문 전송
- 스트리밍 도중 중단
- 세션 3개 생성 후 전환/삭제
- 플랜 모드 ON/OFF 비교
- 모델 변경 후 요청
