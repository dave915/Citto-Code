# 01. 제품 목표와 범위

## 1) 제품 목표
Claude Code CLI를 데스크톱 GUI로 제공해, 비개발자도 대화형 코드 어시스턴트를 사용할 수 있게 한다.

## 2) 핵심 사용자 가치
- 세션 기반 대화
- 실시간 스트리밍 응답 확인
- 파일 첨부를 통한 문맥 제공
- 실행 중단/권한 모드/플랜 모드/모델 선택

## 3) 포함 범위
- 멀티 세션 채팅
- Claude CLI 스트리밍 출력 렌더링
- 파일/폴더 선택
- 파일 첨부 전송
- 실행 중단
- 권한 모드/플랜 모드/모델 선택
- 툴 호출 내역 표시
- 마지막 비용 표시

## 4) 제외 범위
- 로그인 UI
- 서버 백엔드
- 메시지 영구 저장(DB)
- 테스트 러너 내장 UI

## 5) 주요 사용자 시나리오
1. 앱 실행 시 기본 세션 1개 생성
2. 메시지 입력 후 Enter로 전송
3. 스트리밍 텍스트/툴 이벤트 실시간 표시
4. 필요 시 중단 버튼으로 실행 정지
5. 파일 첨부 후 질문 전송
6. 세션 여러 개 생성 및 전환
