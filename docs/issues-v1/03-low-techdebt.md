# 03. Low / Tech Debt

## [LOW] 테스트 자동화 부재
- 상태: Open
- 영역: 품질 관리
- 증상:
  - 단위/E2E 테스트 스크립트 없음
- 영향:
  - 회귀 탐지 늦어짐
- 권장 수정:
  - 최소 E2E: 전송, 중단, 세션 전환, 첨부

## [LOW] 사용되지 않는 의존성 후보
- 상태: Open
- 영역: 빌드/의존성
- 증상:
  - `rehype-highlight` 사용 흔적 없음(현재 스타일 수동)
- 영향:
  - 번들/관리 복잡도 증가
- 권장 수정:
  - 실제 사용 여부 재점검 후 제거 또는 연결

## [LOW] 상태 영속성 없음
- 상태: Open
- 영역: 사용자 경험
- 증상:
  - 앱 재시작 시 세션/대화 소실
- 영향:
  - 사용성 저하
- 권장 수정:
  - 세션 메타/메시지 로컬 저장(persist)
